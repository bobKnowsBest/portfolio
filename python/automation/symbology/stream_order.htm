Description:
Abstract:
This coverage contains selected arcs from the National Hydrogrophy Dataset which was developed by the USGS and EPA. The selected arcs represent the centerlines of wide streams, impoundments, reservoirs, and wetlands as well as the segments of single line streams. An Arc/Info macro was run on the arc segments to calculate the Stralhler stream order for each arc.

Process_Step:
Process_Description:
Original National Hydrography Dataset was provided as 8-digit HUCs in geographic coordinates. This was projected to UTM Zone 15 with the NAD27 datum using Arc/Info 8.0.2. Extraneous arcs were removed from each 8-digit HUC dataset so that only the centerline and single width streams were left. These arcs were then separated into the 11-digit HUCs and a macro to determine the Strahler stream order was run in Arc/Info 8.0.2. The arcs from the 11-digit HUCs were then recombined into the 8-digit HUCs and the 8-digit HUCs were further recombined into a state-wide coverage. This coverage was then converted into a shapefile.
